@par Example Description:

This example describes how to use HT32 BFTM. Both BFTM repetitive and one shot mode are present. Interrupt
and polling is used in this example.

@par Directory Contents:

- BFTM/RepetitiveToggle/main.c               Main program
- BFTM/RepetitiveToggle/ht32fxxxx_it.c       Interrupt handlers

@par Hardware and Software Environment:

- This example can be run on HT32 Series development board.
